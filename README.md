# drone RF dataset
# This repo contains binary files of drones and noise RF magnitudes and signals
# Collected by HackRF One and GNU Radio

<br>

# Params:
Sample rate - 40M<br>
Center freq - 2.445GHz

# Data types:
magnitude (mag) - float<br>
signal (sig) - float
